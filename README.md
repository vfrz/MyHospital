# MyHospital
Basic hospital management app for Android

### Sujet : Logistique de lits d'hôpitaux
```
Vous êtes responsable des lits dans des services hospitaliers.
Vous avez donc plusieurs services, chacun a des lits.
Dans ces lits, vous pouvez avoir des patients (qui ont un nom, sont arrivés et partiront), ou le lit est défectueux ou le lit est en nettoyage ou le lit est vide.
La gestion du temps est virtuelle en ce sens que c'est sur le téléphone que vous indiquez le passage du temps.
Un état coloré est intéressant pour évaluer l'état des lits et la saturation des services.
La durée de nettoyage ou de réparation des lits est à prévoir vous avez une équipe réduite de personnel à cet effet (1 ou 2 personnes).
Vous aurez donc les concepts de service, lit, patient, personnel.
```
